The Mahara Beacon plugin

This is based on the original moodle version of this plugin with modifications for use in the mahara environment.
The Moodle version of this pluign can be found at:  https://git.catalyst-au.net/elearning/moodle-tool_beacon
To add a question, add a new quesiton in the questions folder following one of the existing questions as an example,
 then add an include for your question in processor.php
